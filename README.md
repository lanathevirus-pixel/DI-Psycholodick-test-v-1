# PsycholoD*ick Test — FR/EN avec switch de langue

Ce dépôt contient une page HTML autonome pour le test **PsycholoD*ick** avec :
- interface bilingue FR/EN (header, boutons, Q1),
- calcul automatique des scores,
- données et interprétations actuellement principalement en français.

## Contenu

- `index.html` : toute l'application (HTML + CSS + JavaScript) dans un seul fichier.

## Déploiement sur GitHub Pages

1. Utilise ton dépôt existant ou crée-en un nouveau.
2. Envoie ces fichiers sur la branche `main` :
   - `index.html`
   - `README.md`
3. Dans les paramètres du dépôt, active **GitHub Pages** en choisissant la branche `main` et le dossier `/root`.
4. Ta page sera disponible à une URL du type :
   `https://ton-compte.github.io/nom-du-repo/`.

Aucun build n'est nécessaire : c'est du pur HTML/JS.
