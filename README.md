# PsycholoD*ick Test — Page Web

Ce dépôt contient une page HTML autonome pour le test **PsycholoD*ick** avec calcul automatique des scores.

## Contenu

- `index.html` : toute l'application (HTML + CSS + JavaScript) dans un seul fichier.

## Déploiement sur GitHub Pages

1. Crée un nouveau dépôt sur GitHub (public).
2. Envoie ces fichiers sur la branche `main` :
   - `index.html`
   - `README.md`
3. Dans les paramètres du dépôt, active **GitHub Pages** en choisissant la branche `main` et le dossier `/root`.
4. Quelques instants plus tard, ta page sera disponible à une URL du type :
   `https://ton-compte.github.io/nom-du-repo/`.

Aucun build n'est nécessaire : c'est du pur HTML/JS.
